# MoneyTrack
App para control de gastos personales
## Descripción
MoneyTrack es una aplicación web diseñada para el control de gastos personales, permitiendo a los usuarios registrar, clasificar y analizar sus ingresos y gastos con el fin de mejorar su organización financiera.
## Objetivo
Desarrollar una solución tecnológica que facilite la gestión de las finanzas personales y promueva hábitos financieros saludables en los usuarios.
## Línea de proyecto
Este proyecto se desarrolla bajo la línea:
**Mente Activa — Bienestar Mental y Hábitos Digitales**
Debido a que fomenta hábitos responsables en la gestión del dinero, contribuyendo al bienestar emocional de los usuarios.
## Tecnologías utilizadas
* React (Frontend)
* Node.js (Backend)
* Express
* MongoDB (Base de datos)
* JSON Web Tokens (JWT)
* Figma
## Metodología
El desarrollo del proyecto se realiza bajo la metodología ágil Scrum, mediante Sprints iterativos que permiten la mejora continua del sistema.
## Usuarios del sistema
* Personas que desean controlar sus gastos personales
* Estudiantes
* Trabajadores
## Funcionalidades principales
* Registro de usuarios
* Inicio de sesión
* Registro de ingresos y gastos
* Clasificación por categorías
* Visualización de historial financiero
